A simple Web-Based To-Do App, where the user can enter todo and its discriprtion to make a todo list and display it in the web screen. 

The To-Do List is locally saved to Local Storage; so reload the page we  not will reset the todo list.

Their are respective buttons to add To-Do to the list and delete To-Do from the List.

The Title and the Home Button in the Nav Bar is mapped to "./" OR ROOT and the About is mapped to "./about".
